# Dynamic Programing

Optimised recursion through memoization

### Flow

1. Base case generation
2. Explore all possible valid paths
3. Return based on question

### Patterns

1. Minimum (Maximum) Path to Reach a Target
2. Distinct Ways
3. Merging Intervals
4. DP on Strings
5. Decision Making
